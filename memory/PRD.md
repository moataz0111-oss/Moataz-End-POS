## كيف يعمل النظام:

```
المخزون المركزي (مواد خام: لحم، خبز، بهارات)
        ↓
   [الوصفات تحدد المكونات]
        ↓
المنتجات النهائية (برغر لحم = 150غ لحم + 1 خبز + 5غ بهارات)
        ↓
   [طلب الفرع]
        ↓
خصم تلقائي من المواد الخام حسب الوصفة
```

**مثال عملي:**
- منتج "برغر لحم كامل" له وصفة:
  - 0.15 كيلو لحم صب
  - 1 حبة خبز برغر
  - 5 غرام بهارات
- عند طلب 10 برغر من الفرع:
  - يتم خصم: 1.5 كيلو لحم، 10 خبز، 50 غرام بهارات

---

## نقاط API الجديدة:

| النقطة | الوصف |
|--------|-------|
| `POST /api/finished-products` | إنشاء منتج نهائي مع وصفة |
| `GET /api/finished-products` | جلب المنتجات النهائية مع وصفاتها |
| `PUT /api/finished-products/{id}` | تحديث منتج نهائي |
| `GET /api/raw-materials` | جلب المواد الخام |
| `POST /api/branch-orders` | إنشاء طلب مع خصم تلقائي |
