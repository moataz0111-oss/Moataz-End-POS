# Maestro EGP - نظام إدارة المطاعم المتكامل

## المشكلة الأصلية
بناء نظام إدارة مطاعم متكامل (React/FastAPI/MongoDB) مع بنية متعددة المستأجرين (multi-tenant) يشمل:
- نظام نقاط البيع (POS)
- إدارة المخزون والمشتريات
- نظام التوصيل وإدارة السائقين
- قائمة الزبائن (PWA)
- لوحة تحكم المالك (Super Admin)

## الملفات الرئيسية

```
/app
├── backend/
│   ├── server.py                    # الخادم الرئيسي
│   └── routes/
│       └── drivers_routes.py        # APIs السائقين
└── frontend/
    ├── public/
    │   ├── index.html               # نقطة دخول المسؤولين
    │   ├── menu.html                # نقطة دخول الزبائن (PWA)
    │   ├── manifest-menu.json       # بيان PWA للزبائن
    │   └── sw-customer.js           # Service Worker
    └── src/pages/
        ├── DriverApp.js             # تطبيق السائق
        ├── CustomerMenu.js          # قائمة الزبائن
        ├── Delivery.js              # إدارة التوصيل
        └── Ratings.js               # صفحة التقييمات
```
