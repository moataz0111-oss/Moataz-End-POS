# Maestro EGP - نظام إدارة المطاعم المتكامل

## المشكلة الأصلية
بناء نظام إدارة مطاعم متكامل (React/FastAPI/MongoDB) مع بنية متعددة المستأجرين (multi-tenant).

## الملفات الرئيسية

```
/app
├── backend/
│   ├── server.py                    # الخادم الرئيسي
│   └── routes/
│       └── drivers_routes.py        # APIs السائقين
└── frontend/
    ├── public/
    │   ├── index.html               # نقطة دخول المسؤولين
    │   ├── menu.html                # نقطة دخول الزبائن (PWA)
    │   └── sw-customer.js           # Service Worker
    └── src/
        ├── context/
        │   └── CurrencyContext.js   # إدارة العملة (جديد)
        └── pages/
            ├── DriverPortal.js      # بوابة السائق
            ├── DriverApp.js         # تطبيق السائق
            ├── Settings.js          # الإعدادات (مع تبويب النظام)
            ├── Delivery.js          # إدارة التوصيل
            └── CustomerMenu.js      # قائمة الزبائن
```
