## التحديثات السابقة (21 يناير 2025)

### ✅ إصلاح مشكلة تثبيت تطبيق العملاء (PWA) - جديد!
**المشكلة:** عند تثبيت تطبيق العملاء من iOS Safari، كانت تظهر أيقونة واسم تطبيق السائق بدلاً من تطبيق العملاء.

**السبب:** iOS Safari يخزن أول manifest يجده للنطاق ويتجاهل الباقي عندما يكون `scope` واحد.

**الحل المُنفذ:**
1. **إنشاء مجلد منفصل `/customer-pwa/`** مع:
   - `manifest.json` خاص بتطبيق العملاء مع `scope: "/customer-pwa/"`
   - `index.html` صفحة تثبيت مخصصة للعملاء

2. **تحديث manifest السائق** بإضافة `scope: "/driver"` لفصله

3. **إنشاء Service Worker منفصل** (`customer-sw.js`) لتطبيق العملاء

4. **تحديث Dialog رابط القائمة في Dashboard:**
   - رابط القائمة العادي (للتصفح) + QR أسود
   - رابط تثبيت التطبيق (PWA) + QR برتقالي ⭐
   - الملفات: `/customer-pwa/manifest.json`, `/customer-pwa/index.html`, `/customer-sw.js`

**للاختبار:** يجب على المستخدم:
1. مسح ذاكرة التخزين المؤقت للموقع في Safari (الإعدادات > Safari > محو سجل التصفح والبيانات)
2. فتح الرابط: `{DOMAIN}/customer-pwa/`
3. إضافة للشاشة الرئيسية

## التحديثات السابقة (يناير 2025)

### ✅ نظام المشتريات والمخزون والتصنيع الجديد

#### التدفق الكامل:
```
المورد ← المشتريات (فاتورة+صورة) ← المخزن (مواد خام) ← التصنيع (وصفات) ← الفروع ← الزبون
```

#### الميزات المُنفذة:

**1️⃣ صفحة المشتريات (`/purchases-new`):**
- إضافة وإدارة الموردين
- إنشاء فواتير شراء مع أصناف متعددة
- رفع صورة الفاتورة (إلزامي)
- زر "إرسال للمخزن" ← ينقل المواد تلقائياً
- أرشيف فواتير لكل مورد
- عرض طلبات الشراء من المخزن

**2️⃣ صفحة المخزن والتصنيع (`/warehouse-manufacturing`):**
- **تبويب المخزن:** عرض المواد الخام + زر "إضافة للتحويل"
- **تبويب التصنيع:** 
  - عرض المواد المستلمة من المخزن
  - إنشاء وصفات (منتجات مصنعة) مع المكونات
  - زر **تصنيع** لإنتاج كميات (يخصم المواد تلقائياً)
  - حساب تكلفة الوصفة تلقائياً
- **تبويب الحركات:** واردات وصادرات المخزن
- **تبويب التحويلات:** سجل التحويلات

**3️⃣ صفحة طلبات الفروع (`/branch-orders`):**
- طلب من قسم التصنيع فقط
- دورة حياة الطلب: قيد الانتظار ← موافقة ← شحن ← تسليم
- عند الشحن: خصم من التصنيع
- عند التسليم: إضافة لمخزون الفرع
- عرض مخزون كل فرع

**4️⃣ إعدادات المخزون (في الإعدادات):**
- اختيار نوع المخزون:
  - مخزون مركزي (موزع على الفروع)
  - مخزون منفصل لكل فرع
- الخصم التلقائي عند البيع (تشغيل/إيقاف)
- تنبيهات نقص المخزون (تشغيل/إيقاف)

**5️⃣ حقول جديدة للمنتجات:**
- **تكلفة التغليف:** يُضاف للتكلفة عند البيع سفري/توصيل
- **ربط بمنتج مصنع:** للخصم التلقائي من مخزون الفرع عند البيع

**6️⃣ تكامل POS مع النظام الجديد:**
- حساب تكلفة التغليف للطلبات السفري/التوصيل
- خصم تلقائي من مخزون الفرع عند البيع
- حساب الربح الحقيقي شامل التغليف

## APIs الجديدة

### المشتريات
- `POST /api/suppliers` - إضافة مورد
- `GET /api/suppliers` - جلب الموردين
- `POST /api/purchases-new` - إنشاء فاتورة شراء
- `POST /api/purchases-new/{id}/upload-invoice` - رفع صورة الفاتورة
- `POST /api/purchases-new/{id}/send-to-warehouse` - إرسال للمخزن
- `GET /api/warehouse-purchase-requests` - طلبات الشراء من المخزن

### المخزون (المواد الخام)
- `POST /api/raw-materials-new` - إضافة مادة خام
- `GET /api/raw-materials-new` - جلب المواد الخام
- `POST /api/warehouse-to-manufacturing` - تحويل للتصنيع
- `GET /api/warehouse-transfers` - جلب التحويلات
- `GET /api/warehouse-transactions` - جلب الحركات

### التصنيع
- `GET /api/manufacturing-inventory` - مخزون التصنيع
- `POST /api/manufactured-products` - إضافة منتج مصنع (وصفة)
- `GET /api/manufactured-products` - جلب المنتجات المصنعة
- `POST /api/manufactured-products/{id}/produce` - تصنيع كمية

### طلبات الفروع
- `POST /api/branch-orders-new` - إنشاء طلب فرع
- `GET /api/branch-orders-new` - جلب الطلبات
- `PATCH /api/branch-orders-new/{id}/status` - تحديث الحالة
- `GET /api/branch-inventory/{branch_id}` - مخزون الفرع

### الإعدادات
- `GET /api/inventory-settings` - جلب إعدادات المخزون
- `PUT /api/inventory-settings` - تحديث إعدادات المخزون
- `GET /api/inventory-stats` - إحصائيات المخزون
